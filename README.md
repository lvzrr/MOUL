## INSTALL

Just run this command:
```bash
git clone https://github.com/lvzrr/MOUL ~/.moul && cd ~/.moul && make install && moul && exit
```
## USAGE 
```bash
moul <number of CXX>
```
### EXAMPLE:
For C01 it would be:
```bash
moul 1
```
> [!WARNING] 
> Make sure you are in the root of your git repo
